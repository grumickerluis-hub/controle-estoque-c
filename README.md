# Sistema de Controle de Estoque

Projeto desenvolvido durante a disciplina de Laboratório de Programação da graduação em Ciência da Computação.

## Objetivo

Desenvolver um sistema de controle de estoque utilizando a linguagem C, aplicando conceitos fundamentais de programação, modularização e organização de código.

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Registro de vendas
- Controle de estoque
- Relatórios de vendas

## Tecnologias utilizadas

- Linguagem C
- VS Code

## Conceitos aplicados

- Funções
- Vetores
- Estruturas de decisão
- Estruturas de repetição
- Modularização
- Organização de código
